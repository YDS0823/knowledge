# Markdown 与图片使用示例

## 文字格式

普通段落，**加粗**，*斜体*，~~删除线~~，`行内代码`。

> 引用文字，适合记录重要提示。

## 列表

- 知识点 A
- 知识点 B
  - 子知识点 B-1
  - 子知识点 B-2

## 表格

| 名称 | 说明 |
|------|------|
| 示例A | 这是一个说明 |

## 代码块

```python
def hello(name):
    print(f"Hello, {name}!")
```

## 图片

将图片放到 `docs/images/` 目录，用以下方式引用（点击可放大）：

```markdown
![描述](docs/images/文件名.png)
```
