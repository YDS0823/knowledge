### [42. 接雨水](https://leetcode.cn/problems/trapping-rain-water/) - 双指针

给定 `n` 个非负整数表示每个宽度为 `1` 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

 

**示例 1：**

![img](https://assets.leetcode.cn/aliyun-lc-upload/uploads/2018/10/22/rainwatertrap.png)

```
输入：height = [0,1,0,2,1,0,1,3,2,1,2,1]
输出：6
解释：上面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，可以接 6 个单位的雨水（蓝色部分表示雨水）。 
```

**示例 2：**

```
输入：height = [4,2,0,3,2,5]
输出：9
```

思路：维护左最高和右最高，然后遍历左柱子或者右柱子，与左最高或者右最高比较，每次sum只增加一个横坐标的，局部累加到总和。

```c++
class Solution {
public:
    int trap(vector<int>& height) {
        int leftmax=0,rightmax=0,sum=0;
        int left=0,right=height.size()-1;
        while(left<=right){
            if(leftmax<rightmax){
                if(height[left]>leftmax){//当前柱子比左最高还高，水往左流出了
                    leftmax=height[left];
                }else{//否则水往右流，又因为rightmax>leftmax，所以水不会流出去，至少能存住一个横坐标
                    sum+=(leftmax-height[left]);
                }
                left++;
            }else{//同理
                if(height[right]>rightmax){
                    rightmax=height[right];
                }else{
                    sum+=(rightmax-height[right]);
                }
                right--;
            }
        }
        return sum;
    }
};
```



### [124. 二叉树中的最大路径和](https://leetcode.cn/problems/binary-tree-maximum-path-sum/)-二叉树

二叉树中的 **路径** 被定义为一条节点序列，序列中每对相邻节点之间都存在一条边。同一个节点在一条路径序列中 **至多出现一次** 。该路径 **至少包含一个** 节点，且不一定经过根节点。

**路径和** 是路径中各节点值的总和。

给你一个二叉树的根节点 `root` ，返回其 **最大路径和** 。

 

**示例 1：**

![img](https://assets.leetcode.com/uploads/2020/10/13/exx1.jpg)

```
输入：root = [1,2,3]
输出：6
解释：最优路径是 2 -> 1 -> 3 ，路径和为 2 + 1 + 3 = 6
```

**示例 2：**

![img](https://assets.leetcode.com/uploads/2020/10/13/exx2.jpg)

```
输入：root = [-10,9,20,null,null,15,7]
输出：42
解释：最优路径是 15 -> 20 -> 7 ，路径和为 15 + 20 + 7 = 42
```

这题不需要返回路径，返回最大值就行，所以用递归，局部用整条链路，网上递归时只保留最大的单边的路径

```c++
/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode() : val(0), left(nullptr), right(nullptr) {}
 *     TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
 *     TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
 * };
 */
class Solution {
private:
    int maxsum=INT_MIN;//因为还有负值，所以要负最小
public:
    int maxgain(TreeNode* node){
        if(node==nullptr)return 0;
        int leftgain=max(maxgain(node->left),0);//分别记录左右两个子树的最大单路价值
        int rightgain=max(maxgain(node->right),0);

        if(leftgain+rightgain+node->val > maxsum){ //如果当前节点加上两个子树的值更大，则换
            maxsum=leftgain+rightgain+node->val;
        }

        return node->val+max(leftgain,rightgain);//返回当前节点的单路价值
    }
    int maxPathSum(TreeNode* root) {
        maxgain(root);
        return maxsum;
    }
};
```



### [51. N 皇后](https://leetcode.cn/problems/n-queens/) - 回溯

按照国际象棋的规则，皇后可以攻击与之处在同一行或同一列或同一斜线上的棋子。

**n 皇后问题** 研究的是如何将 `n` 个皇后放置在 `n×n` 的棋盘上，并且使皇后彼此之间不能相互攻击。

给你一个整数 `n` ，返回所有不同的 **n 皇后问题** 的解决方案。

每一种解法包含一个不同的 **n 皇后问题** 的棋子放置方案，该方案中 `'Q'` 和 `'.'` 分别代表了皇后和空位。

 

**示例 1：**

![img](https://assets.leetcode.com/uploads/2020/11/13/queens.jpg)

```
输入：n = 4
输出：[[".Q..","...Q","Q...","..Q."],["..Q.","Q...","...Q",".Q.."]]
解释：如上图所示，4 皇后问题存在两个不同的解法。
```

**示例 2：**

```
输入：n = 1
输出：[["Q"]]
```

很简单，就是看看当前格能不能放，然后用回溯去便利所有状态

```c++
class Solution {
public:
    void backtrack(vector<vector<string>>& res,vector<string>& board,
    vector<bool>& colmn,vector<bool>& diag, vector<bool>& un_diag,int row,int n){
        if(row==n){
            res.push_back(board);
            return ;
        }
        for(int col=0;col<n;col++){
            if(colmn[col] || diag[col-row+n-1] || un_diag[row+col]){//这里的对角线，就是当前所在的格子位于第几条对角线上。
                continue;
            }
            board[row][col]='Q';//试试放
            colmn[col]=diag[col-row+n-1]=un_diag[row+col]=true;
            backtrack(res,board,colmn,diag,un_diag,row+1,n);
            board[row][col]='.';//回溯，变回原来的状态
            colmn[col]=diag[col-row+n-1]=un_diag[row+col]=false;//状态也恢复   
        }
    }
    vector<vector<string>> solveNQueens(int n) {
        vector<bool>colmn(n,false);//记录纵项是否已经有皇后,由于递归的是行，所以不用管行，只便利列
        vector<bool>diag(2*n-1,false);//记录对角线是否有皇后
        vector<bool>un_diag(2*n-1,false);//记录反对角线是否有皇后
        vector<string>board(n,string(n,'.'));//初始化棋盘，一个n行，每行n个点的string的棋盘
        vector<vector<string>>res;//答案格式，照抄题目即可
        backtrack(res,board,colmn,diag,un_diag,0,n);//开始回溯,第0行开始，直到有n个皇后
        return res;
    }
};
```
